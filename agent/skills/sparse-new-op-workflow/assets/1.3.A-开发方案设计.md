### 函数拆分

```cpp

static aclsparseStatus_t Launch{Op}Kernel(...);

    auto status = Validate{Op}Params(...);
    if (status != ACL_SPARSE_STATUS_SUCCESS) return status;
    return Launch{Op}Kernel(...);
}
```
