## 7. Host 侧设计

### 7.1 函数拆分

```cpp
static aclsparseStatus_t Validate{Op}Params(...);   // 参数校验
static aclsparseStatus_t Launch{Op}Kernel(...);     // 描述符解析 + tiling 计算 + kernel launch

aclsparseStatus_t aclsparse{Op}(...) {
    // 边界快速返回
    if (handle == nullptr) return ACL_SPARSE_STATUS_HANDLE_IS_NULLPTR;
    auto status = Validate{Op}Params(...);
    if (status != ACL_SPARSE_STATUS_SUCCESS) return status;
    return Launch{Op}Kernel(...);
}
```
